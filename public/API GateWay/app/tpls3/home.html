PCN & CAD API Gateway

API Gateway ，一个出现在企业级IT系统边界上的面向API、串行集中式的强管控服务。

 

动态路由

服务路由功能为服务和用户之间交互提供路由机制，当收到服务请求消息，通过解析目的服务信息分析服务请求，分发消息到最终目的地。 并在流量监测的基础之上，得到用户访问情况。

API Gateway 可以让您与您的API进行交互，来创建或编辑APIs, 用户 和 组件。

服务监控

服务监控包括健康检查、性能监控。

健康检查主要包括监控用户访问的频繁程度。

性能监控是对开放平台的性能状态进行实时监控。

水平拓展

API网关的实际应用场景下存在高并发的场景，因此平台不仅实现开放能力平台的业务功能，还保证系统的高并发、高可用和高可靠。

API网关集群采用了负载均衡，缓存机制，消息队列，池化技术来保证系统高并发情境下的性能。
